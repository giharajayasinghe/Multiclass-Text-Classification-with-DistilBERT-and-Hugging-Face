# Multiclass-Text-Classification-with-DistilBERT-and-Hugging-Face
Fine-tuned distilbert-base-uncased using Hugging Face Transformers on 5,000 samples from the yelp_review_full dataset for multiclass text classification. The model predicts Yelp review ratings from 1 to 5 stars using NLP preprocessing, tokenization, training, and evaluation.Face transformers and datasets libraries.
